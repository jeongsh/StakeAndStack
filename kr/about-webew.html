위뷰에 대해

우리는 웹 3 빌더입니다. 우리는 웹을 뒤집어 놓습니다.

we⋮bew라는 이름은 웹을 변화시키고 새로운 패러다임을 만들기 위한 우리의 임무에서 유래되었습니다. we와 bew 사이의 세 개의 점은 숫자 3을 나타냅니다. Web 3를 반대로 쓰면 3 bew가 되며, 숫자 3은 점으로 표현됩니다. Web 3 빌더로서, 우리는 웹 3 생태계 내의 두 가지 사업 영역에 초점을 맞추고 있습니다: 인프라 서비스와 웹 3 서비스.

우리의 비전

우리의 비전은 "모두가 블록체인을 검증하고 보안을 유지하며, 모두가 웹 3 커뮤니티의 소유자가 되는 세상"입니다. 이 목표를 달성하기 위해 우리는 사용자 친화적인 제품과 서비스를 개발하고 있습니다.

사업 영역

1. 웹 3 인프라

  • 밸리데이터(PoS 밸리데이터): 스테이크앤스택

  • 우리는 다양한 네트워크의 밸리데이터를 운영하고, 위임자들이 거버넌스에 적극 참여할 수 있도록 인프라와 연구를 제공할 계획입니다.

  • 밸리데이터 원스톱 솔루션

  • 기술적 지식이 없어도 누구나 쉽게 밸리데이터를 운영할 수 있도록 하는 밸리데이터 원스톱 솔루션(PaaS).

2. 웹 3 서비스

  • 멤버십 커뮤니티 플랫폼: ⋮bew

  • 뷰는 멤버십과 구독권이 토큰화되어 같은 관심사를 가진 사용자들이 함께 커뮤니티를 성장시킬 수 있는 크리에이터 경제 기반의 웹 3 커뮤니티 플랫폼입니다.

  • 우리는 멤버십 토큰을 새로운, 신뢰할 수 있는, 재판매할 수 있는 자산으로 제공합니다. 크리에이터들은 토큰 판매를 통해 유동성을 얻을 수 있으며, 사용자들은 자신의 멤버십 토큰에 대해 리뷰를 작성하고, 홍보하고, 가치를 상승시킬 수 있습니다. 수익은 커뮤니티 회원들 사이에 분배되어 성장과 발전을 촉진합니다.

  • 사용자 혜택
    크리에이터: 수익, 편의성, 타겟팅된 관객
    회원: 수익 창출, 편의성, 재능 있는 크리에이터 발견
    광고주: 효율적인 타겟팅 및 경제적인 광고
